Phase 0
需求定義
工具名稱 *必填
用 2–4 個字命名這個工具
GitHub Repo 名稱
將作為 GitHub 專案名稱，僅限英文、數字、連字號。會根據工具名稱自動產生，可自行修改
工具概述 *必填
請描述你想像中的工具內容，AI 會根據這段描述自動拆解並填入後續欄位
參考檔案 （選填）
上傳現有的 Excel、CSV、PDF、PPT 或文字檔作為 AI 填表的參考資料。例如：以前用 Excel 做的 thermal report，AI 會參考裡面的欄位和格式來填寫規格。

📎 點擊上傳參考檔案

支援 .xlsx、.csv、.pdf、.pptx、.txt、.json、.md（最多 3 個檔案）
